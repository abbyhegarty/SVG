Scale

In D3, scales are functions that set definable parameters. Scales have two parts: an input domain and an output domain. can add an a data value and the function will return back an scaled output value.

The main way Murray discussed scaling was in realtion to scatter plots. Visually, scaling the data makes a scatter plot easier to look at relationship between data points in a set. Additionally, scales can be used visually to manipulate how data is shown. For example, scatterplots can use an x and y scale or an inverted y scale to produce two drastically different scatterplots.
